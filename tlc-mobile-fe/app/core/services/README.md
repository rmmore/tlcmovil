Descripcion:

1) profile-services.js : En este js se encuentra el objeto factory el cual realiza la llamada al servicio de perfil.
2) cas-services.js : En este js se encuentra la validacion del servicio de token de cas al igual que al relizar el logout envia dicho token su eliminicacion
3) storage-services.js: En este js se obtiene y almacena los token de la aplicacion.
4) unload-services.js: En este js se encuentran las funciones de unload de la aplicacion.
